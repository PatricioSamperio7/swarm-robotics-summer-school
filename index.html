Program Overview

Duration

2 Weeks (10 Business Days)

Format

Hybrid (Theory + Applied Project)

Total Hours

25 Hours Intensive (12.5h/week)

Tech Stack

C++ and ROS 2 Framework

This course focuses on robotic swarms, distributed systems, robot-to-robot communication, IoT integration, and Edge Computing using scalable architectures.

Technical Syllabus

Week 1: Intensive Technical Training

  • Introduction to Swarm Robotics & Complex Systems
  • Multi-robot vs. Robotic Swarms differences
  • Self-organization principles and distributed control
  • Advanced ROS 2: Nodes, Topics, Services, and Actions
  • Namespaces and scalability for multi-robot fleets
  • Robot-to-robot communication in distributed environments
  • DDS & Quality of Service (QoS) Configuration
  • IoT and Edge Computing for Swarm Intelligence
  • Hardware Architecture for Cooperative Agents
  • C++ Implementation of Swarm Behaviors

Week 2: Applied Engineering Project

Real-world Problem Solving Tracks:

  • Track A: Cooperative Industrial Infrastructure Inspection
  • Track B: Cooperative Light Load Transport Coordination
  • Track C: Swarm Mapping (Distributed Exploration)
  • Track D: Mobile IoT Distributed Monitoring (Edge Nodes)
  • Full System Implementation, Debugging, and Validation
  • Technical Documentation and Final Swarm Demonstration
