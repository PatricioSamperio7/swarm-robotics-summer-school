Swarm Robotics Summer School 2026

Design and Implementation of Distributed Robotics Swarms using ROS 2

Hosted at Facultad de Ingeniería Mecánica y Eléctrica
Universidad Autónoma de Nuevo León, México.

About the Summer School

This practitioner-oriented program focuses on real-world implementation of swarm robotics systems using ROS 2. Participants will explore distributed autonomy, multi-agent coordination, decentralized communication architectures and robotic deployment strategies aligned with industrial and research environments. The objective is to bridge academic robotics education with practical engineering execution.

Program Highlights

Week 1 — Foundations

ROS 2 middleware architecture, DDS communication, Linux robotics environment configuration, distributed system fundamentals, simulation workflows using Gazebo.

Week 2 — Swarm Algorithms

Consensus algorithms, formation control, behavior-based robotics, multi-agent localization strategies.

Week 3 — Implementation

Hardware integration, sensor fusion, robot communication protocols, collaborative testing environments.

Final Demonstration

Distributed robotic swarm deployment, team validation, technical presentation and evaluation.
