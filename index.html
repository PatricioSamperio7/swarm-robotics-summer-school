Value Proposition

Core Competencies

  • Development of ROS 2 distributed architectures.
  • Implementation of multi-robot coordination algorithms.
  • Deployment of swarm logic on physical embedded hardware.
  • Optimization of DDS/QoS communication layers.

Technical Deliverables

  • Functional ROS 2 multi-agent repository.
  • System technical documentation.
  • Performance demonstration video.
  • Official IEEE Participation Certificate.

Target Audience

  • Senior undergraduate students.
  • Graduate researchers (MSc/PhD).
  • Autonomous systems developers.
  • Embedded systems engineers.

Academic Program

Module 01: Foundations

Principles of Swarm Robotics, self-organization theory, and analysis of emergent behaviors in complex systems.

Module 02: ROS 2 Architecture

Distributed nodes, namespaces management, and scalable communication for multi-robot fleets.

Module 03: DDS & Networking

Configuration of Quality of Service (QoS) profiles, DDS middleware tuning, and R2R communication.

Module 04: Hardware Integration

Embedded ROS 2 deployment, sensor fusion, and physical hardware-software synchronization.

Module 05: Cooperative Control

Implementation of decentralized decision-making algorithms and swarm coordination logic.

Final Project

Integration, validation, and physical deployment of a functional swarm robotics system.

Application Process

Selection Criteria

  • Academic alignment with robotics research.
  • Technical proficiency in C++ or Python.
  • Quality of the statement of purpose.

Technical Requirements

  • Laptop with Ubuntu 22.04/24.04: Mandatory as the native environment for ROS 2 Humble/Jazzy deployment.
  • Basic Programming: Understanding of OOP in C++ or Python.

FAQ

Is prior ROS 2 experience required?
Basic knowledge is recommended, but Module 02 provides the necessary foundations.

Is Ubuntu Linux mandatory?
Yes. ROS 2 deployment and DDS communication require a native Linux environment for full compatibility with laboratory hardware.

Is accommodation provided?
No. Participants must arrange their own lodging in Monterrey.
